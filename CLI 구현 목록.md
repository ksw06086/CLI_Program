# 미구현 목록
- 파일을 복사하는 명령어 만들기
- 경로를 지정하면 하위의 모든 폴더와 파일을 지우는 명령어 만들어보기(rimraf 패키지 참조)
- 데이터베이스와 연동해 가계부 만들어보기(시퀄라이즈를 세팅하고 나서 실행 파일 첫 부분에 sequelize.sync로 데이터베이스를 연동한 후 명령어를 수행하면 됨)
