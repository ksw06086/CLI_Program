# CLI 프로그램
- 터미널에서 돌아가는 프로그램 만듬(ex> nodemon / npm 등 명령어 입력하면 실행되게)

# process.argv
- npx cli = node index.js
- node의 경로, index.js의 경로를 보여줌
- 명령어 뒤에 옵션을 더 붙이면 그게 process.argv로 값으로 넘어감
